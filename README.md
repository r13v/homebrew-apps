# homebrew-apps

Homebrew tap for [r13v](https://github.com/r13v)'s apps.

## Usage

```sh
brew tap r13v/apps
brew install <formula>
```

## Formulae

- `plexus` — code-graph CLI: callers, neighbors, paths, communities across Go/Python/JS/TS repos. Source: [r13v/plexus](https://github.com/r13v/plexus).
